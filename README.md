## 🌟 Overview

**GladiatorGame** merges dungeon-crawling elements with survival mechanics. The core gameplay loop focuses on mechanical simplicity and strategic movement within a constrained environment.

* **Genre:** 2D Action / Survival / Roguelite
* **Core Objective:** Survive consecutive waves of enemies within a dynamic arena.
* **Inspirations:** *Vampire Survivors*, *Brotato*, and classic arcade dungeon crawlers.

---

## ✨ Key Features

Gladiator is designed with a focus on user efficiency and professional output. By utilizing its core processing engine, users can expect the following benefits:

* ⚔️ **Intuitive Combat:** Designed for accessibility and "pick-up-and-play" mechanics that reward timing over complex inputs.
* 🏟️ **Arena Logic:** Features environmental hazards and layout transitions that force players to adapt their positioning constantly.
* 👾 **Scalable Difficulty:** Enemy patterns and density evolve as the session progresses, testing the player's endurance and tactical movement.

## 🎮 How to Play

### Installation & Running
1.  Navigate to the [Releases](https://github.com/DavydGladius/Gladiator/releases) page.
2.  Download the latest version of the `.exe` file.
3.  Launch the executable directly—**no installation or external dependencies required.**

### Controls
| Action | Key / Input |
| :--- | :--- |
| **Move Gladiator** | `W` `A` `S` `D`|
| **Aim / Attack** | `Mouse Movement` / `Left Click` |
